PROJECT TITLE: ANALYSIS OF THE VARIATIONS IN SALARIES AMONG WORKERS¶

This project analyzes the various factors affecting the salary 
ranges in workers ranging from location, to theie level of expertise, the year, company size and lots more. 

OBJECTIVES;
1. Clean and thouroughly analyze the dataset 
2. Analyze some important trends in the dataset.
3. Answer data driven questions and showcase answers using charts.
